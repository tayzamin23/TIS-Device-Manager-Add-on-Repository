# TIS Device Manager (Custom)

This repository provides a clean foundation for building a custom TIS Device Manager.

It includes:

- UDP Sniffer for port 6000  
- Simple scanner using broadcast  
- Basic TIS protocol parser (to be refined)  
- Optional FastAPI API server  
- Dockerfile for running in Home Assistant Add-on or standalone  
- Add-on manifest for Home Assistant (local custom add-on)
